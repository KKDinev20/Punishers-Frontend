# Punishers-Frontend


## Run locally
1. Install Node.js 
```bash
 https://nodejs.org/en
```
2. Clone the project

```bash
 git clone https://github.com/KKDinev20/Punishers.git
```
3. Go to the project directory

```bash
 cd alternative
```

4. Install react components

```bash
 npm install 
```     

5. Run the project
```bash
 npm start
```
